List 심화
=================================
```python
names=['A', 'B', 'C']
print('A' in names) #True
print('D' in names) #False

nums = [1, 100, 12312312]
print(len(nums))#3
nums.reverse()
print(nums) #[12312312, 100, 1]

al=['A', 'B', 'C', 'D']
print(len(al)) #4
al.append('E') #'E' 추가
print(al) #['A', 'B', 'C', 'D', 'E']
del(al[0]) #al[0] 삭제
print(al) #['B', 'C', 'D', 'E']
```
