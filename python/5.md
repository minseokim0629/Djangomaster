List 심화
=================================
```python
names=['A', 'B', 'C']
print('A' in names) #True
print('D' in names) #False

nums = [1, 100, 12312312]
print(len(nums))#3
nums.reverse()
print(nums) #[12312312, 100, 1]

al=['A', 'B', 'C', 'D']
print(len(al)) #4
al.append('E') #'E' 추가
print(al) #['A', 'B', 'C', 'D', 'E']
del(al[0]) #al[0] 삭제
print(al) #['B', 'C', 'D', 'E']
```

Container&Loop
===============================
 ```python
 members = ['egoing', 'leezche', 'graphittie']
 i=0
 while i<len(members):
     print(members[i])
     i=i+1
```

```python
 members = ['egoing', 'leezche', 'graphittie']
 for member in members:
     print(member)
 ```
 
 ```python
 for item in range(5,11): #5~10출력
     print(item)
 
 for item in range(5): #0~4출력 0부터 시작
     print(item)
 
 for item in [0,1,2,3,4]: #이런식으로도 가능 0~4출력
     print(item)
 ```
